RADAR WAR – Matekkal lődd le az ellenséget

Egy minimalista, full-screen, valós idejű rakétavédelmi játék HTML5/JS-ben.

A radar pásztáz. Az ellenség közeledik. Amikor elkapod a sugaraddal → felvillan a szorzás (pl. 127 × 84). Ha gyorsan és helyesen beírod az eredményt → sárga rakéta indul, és pontosan követi a mozgó célpontot. Találat = robbanás + pont. Hibázol vagy lassú vagy → ellenség kirepül → -1 élet.

Készülj fel: az elején könnyű… aztán pokol lesz.
